# SEO Setup Guide for Prime Axis Capitals

## 🚀 Quick Setup to Get Your Site Indexed

### 1. Google Search Console Setup
1. Go to [Google Search Console](https://search.google.com/search-console)
2. Add your property: `https://primeaxiscapital.in`
3. Choose "URL prefix" method
4. Copy the verification meta tag and replace the placeholder in `src/app/layout.tsx`
5. Or use the HTML file method: download the verification file and place it in `public/` folder

### 2. Submit Sitemap
- Sitemap URL: `https://primeaxiscapital.in/sitemap.xml`
- Submit it in Google Search Console under "Sitemaps" section

### 3. Bing Webmaster Tools
1. Go to [Bing Webmaster Tools](https://www.bing.com/webmasters)
2. Add your site: `https://primeaxiscapital.in`
3. Verify ownership
4. Submit sitemap: `https://primeaxiscapital.in/sitemap.xml`

### 4. Request Indexing
- Use "Request Indexing" in Google Search Console for your homepage
- This tells Google to crawl your site immediately

### 5. Social Media Presence
- Create business pages on Facebook, LinkedIn, Instagram
- Add links back to your website
- This helps search engines discover your site

## 📈 Expected Timeline
- **24-48 hours**: Site appears in search results
- **1-2 weeks**: Full indexing of all pages
- **2-4 weeks**: Improved rankings with local keywords

## 🔍 Test Your SEO
- Search: `site:primeaxiscapital.in` to see indexed pages
- Search your target keywords to check rankings
- Use tools like SEMrush or Ahrefs for monitoring
